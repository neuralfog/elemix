# Elemix app scaffold

Scaffold a new elemix app from the [official template](https://github.com/neuralfog/elemix-template).

```sh
npm create elemix-app
# or
pnpm create elemix-app
# or
npx create-elemix-app my-app
```

Pass a directory name to skip the prompt:

```sh
npm create elemix-app my-app
```

Then:

```sh
cd my-app
pnpm install
pnpm dev
```
